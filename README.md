# Система управления геометрическими фигурами (С++)
## Сборка и запуск
Для сборки проекта требуется установленный CMake и компилятор C++.

### Проверить наличие CMake:
```
cmake --version
```

### Сборка проекта:
Из директории проекта выполните:
```
mkdir build
cd build
cmake ..
cmake --build .
```

После успешной сборки исполняемый файл будет находиться в папке build.

### Запуск приложения:
```
./GeometrySystem
```

### Если требуется пересобрать проект заново:
```
cmake --build . --clean-first
```
